Description
This project is a responsive landing page dedicated to the legendary Indonesian hero, Gatot Kaca. It showcases his legendary powers and history using a clean, modern design inspired by Indonesian traditional colors.

Key Features
-Hero Section: A cinematic header featuring the famous quote "Muscle of Wire, Bone of Iron."
-Power Showcase: Responsive cards using Bootstrap 5 to display his abilities (Flight, Invulnerability, and Strength).
-Responsive Design: Fully optimized for both desktop and mobile viewing.
-Custom Styling: A unique color palette of Gold and Deep Red to reflect the character's royalty and strength.

Technologies Used
-HTML5
-CSS3 (Custom styling and layout)
-Bootstrap 5 (For the responsive grid system and components)

How to Run
-Ensure you have an internet connection (to load the Bootstrap CDN).
-Open the index.html file in any modern web browser (Chrome, Firefox, Edge, etc.).
